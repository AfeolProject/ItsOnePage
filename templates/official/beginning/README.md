# The Beginning

The Beginning is the first official ItsOnePage template.

It is a lightweight, responsive and privacy-friendly starting point for a one-page website.

## Features

- Static HTML and CSS
- No build process
- No framework
- No tracking
- No advertisements
- Responsive desktop, tablet and mobile layouts
- MIT License
